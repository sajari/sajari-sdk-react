
## FieldDictionary

The FieldDictionary class is used to map fields in your data to the required fields to display in the UI. In previous versions, this mapping was done via the `Results` component. Now it's provided via the `fields` property into the `SearchProvider` instead.

```js
import { FieldDictionary } from '@sajari/react-search-ui';
```
