# Quick setup

This is a guide for getting a basic search interface up and running using the React SDK.

## Dependencies

Add the required imports for this basic example:

```jsx
import {
  Provider,
  Pipeline,
  Values,
  NoTracking,
  Response,
  Input,
  Results,
  Summary,
  Paginator
} from "@sajari/sdk-react";
```

## Pipeline

All search queries go through a [Pipeline](/controllers/pipeline) controller, so this must be created first:

```jsx
const pipeline = new Pipeline(
  {
    project: "project-id",
    collection: "collection-name"
  },
  "pipeline-name",
  new NoTracking()
);
```

**ðŸš¨ Note**

- Replace `project-id` and `collection-name` with the correct details from the [credentials](https://www.sajari.com/console/collection/credentials) section of the Sajari Console.
- Replace `pipeline-name` with the pipeline name to use for querying.

## Values

Search parameters are passed via the [Values](/controllers/values) controller. Create a new instance and set some defaults:

```jsx
import { Provider, Pipeline, Values, NoTracking } from "@sajari/sdk-react";

const values = new Values({
  resultsPerPage: 5,
  page: 1
});
```

## Provider

All of the search components from the SDK must be within a context [Provider](components/provider). The only required prop is `search` which takes the `pipeline` and `values` created previously.

```jsx
<Provider search={{ pipeline, values }}>
  {/* Your application code goes here */}
</Provider>
```

## Input & Results

Now you need to capture input and display some results using the [`Input`](/components/Input), [`Response`](/components/Response), [`Summary`](/components/Summary), [`Results`](/components/Results), and [`Paginator`](/components/Paginator) components.

```jsx
<Provider search={{ pipeline, values }}>
  <Input />
  <Response>
    <Summary />
    <Results />
    <Paginator />
  </Response>
</Provider>
```

### Example

Now you should have a basic search interface working. For more advanced interfaces, see the [examples](/#examples). Here's the complete example:

```jsx
import React from "react";
import {
  Provider,
  Pipeline,
  Values,
  NoTracking,
  Response,
  Input,
  Results,
  Summary,
  Paginator
} from "@sajari/sdk-react";

export default () => {
  const pipeline = new Pipeline(
    {
      project: "sajariptyltd",
      collection: "sajari-com"
    },
    "website",
    new NoTracking()
  );

  const values = new Values({
    resultsPerPage: 5,
    page: 1
  });

  return (
    <Provider search={{ pipeline, values }}>
      <Input />
      <Response>
        <Summary />
        <Results />
        <Paginator />
      </Response>
    </Provider>
  );
};
```
